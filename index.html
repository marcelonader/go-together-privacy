Política de Privacidade

Última atualização: março de 2026

1. Quem somos

O Go Together é um aplicativo para organização colaborativa de viagens, desenvolvido por Marcelo Massayuki Nader, com sede no Brasil.

Contato para dúvidas, solicitações e exercício de direitos LGPD: marcelo.m.nader@gmail.com

2. Dados que coletamos

Dados coletados automaticamente via Google Sign-In:

  • Nome completo, endereço de e-mail e URL da foto de perfil
  • Identificador único de usuário (UID Firebase)

Dados inseridos pelo usuário para uso no grupo de viagem:

  • Dados de viagens (destinos, datas, roteiros, despesas, checklists, responsabilidades, logística)
  • Nome, e-mail e foto compartilhados com os demais membros ao entrar em uma viagem

Dados sensíveis opcionais (seção "Minhas Informações" do grupo):

Os campos abaixo são totalmente opcionais e classificados como dados sensíveis conforme o art. 11 da LGPD. O preenchimento não é necessário para usar o aplicativo.

  • Número e validade do passaporte — documento de identificação
  • Restrições alimentares — podem revelar condições de saúde
  • Plano de saúde — dado de saúde
  • Seguro viagem — dado financeiro relacionado à saúde
  • Nome e telefone de contato de emergência — dados pessoais de um terceiro que não utiliza o aplicativo. Ao cadastrar essas informações, você declara ter obtido o consentimento dessa pessoa.

Esses dados sensíveis são estritamente privados: apenas você tem acesso ao seu próprio perfil. Outros membros da viagem não podem visualizá-los.

3. Como usamos seus dados

  • Para autenticar você e exibir seu perfil no grupo de viagem
  • Para salvar e sincronizar os dados das viagens entre os membros do grupo
  • Para gerar sugestões de checklist, roteiro e informações de destino por meio de inteligência artificial (Google Gemini) — os dados enviados para esse fim são apenas o nome do destino e as datas da viagem, nunca dados pessoais identificáveis
  • Não vendemos, alugamos nem compartilhamos seus dados com terceiros para fins comerciais

4. Onde os dados ficam armazenados — sub-processadores

Seus dados são processados pelos seguintes serviços terceiros, todos com políticas de privacidade próprias:

  • Google Firebase (Firestore e Authentication) — armazenamento e autenticação, servidores nos Estados Unidos
  • Google Gemini API — geração de sugestões por IA; recebe apenas nome do destino e período da viagem
  • Render.com — servidor intermediário que encaminha requisições à API Gemini, localizado nos Estados Unidos

Todos os dados em trânsito são protegidos por criptografia TLS. Não vendemos nem compartilhamos dados com terceiros para fins comerciais.

5. Por quanto tempo mantemos os dados

Seus dados são mantidos enquanto você tiver uma conta ativa no app. Ao excluir sua conta, todos os dados pessoais são apagados permanentemente — incluindo dados sensíveis de perfil em todas as viagens das quais você fazia parte.

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados — disponível no app em Perfil → Exportar meus dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão de dados — disponível no app em Perfil → Excluir conta
  • Revogar o consentimento a qualquer momento (ao recusar os termos, sua sessão é encerrada)
  • Portabilidade dos dados — o arquivo exportado contém todos os seus dados em formato JSON

Para exercer qualquer desses direitos ou enviar solicitações relacionadas à LGPD, entre em contato: marcelo.m.nader@gmail.com

Respondemos a solicitações em até 15 dias úteis.

7. Transferência internacional

Seus dados são transferidos para servidores nos EUA (Google Firebase, Google Gemini, Render.com). Essa transferência é necessária para o funcionamento do serviço e ocorre com base no seu consentimento explícito, conforme o art. 33, inciso I, da LGPD.

8. Inteligência Artificial

O Go Together utiliza o Google Gemini para gerar sugestões de checklist, atividades de roteiro e informações sobre destinos. Essas sugestões são geradas automaticamente e podem conter imprecisões. Você é responsável por verificar as informações antes de utilizá-las. Dados pessoais identificáveis nunca são enviados para a API de IA.

9. Alterações nesta política

Caso esta política seja alterada de forma significativa, você será notificado no app na próxima abertura e deverá aceitar os novos termos para continuar usando o serviço.
