Política de Privacidade

Última atualização: março de 2026

1. Quem somos

O Go Together é um aplicativo para organização colaborativa de viagens, desenvolvido por Marcelo Massayuki Nader, com sede no Brasil.

Contato: marcelo.m.nader@gmail.com

2. Dados que coletamos

  • Nome, endereço de e-mail e foto de perfil, obtidos via login com Google
  • Dados de viagens criadas ou compartilhadas no app (destinos, datas, roteiros, despesas, checklists)
  • Preferências salvas localmente no dispositivo

3. Como usamos seus dados

  • Para autenticar você e exibir seu perfil no grupo de viagem
  • Para salvar e sincronizar os dados das viagens entre os membros do grupo
  • Não vendemos, alugamos ou compartilhamos seus dados com terceiros para fins comerciais

4. Onde os dados ficam armazenados

Os dados são armazenados no Google Firebase (Firestore e Authentication), com servidores localizados nos Estados Unidos. O Google é certificado nos principais padrões internacionais de segurança.

5. Por quanto tempo mantemos os dados

Seus dados são mantidos enquanto você tiver uma conta ativa no app. Ao excluir sua conta, todos os dados são apagados permanentemente.

6. Seus direitos (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), você tem direito a:

  • Confirmar a existência de tratamento dos seus dados
  • Acessar seus dados
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão de dados — disponível diretamente no app em Perfil → Excluir conta
  • Revogar o consentimento a qualquer momento

Para exercer qualquer desses direitos, entre em contato: marcelo.m.nader@gmail.com

7. Transferência internacional

Seus dados são transferidos para servidores nos EUA (Google Firebase). Essa transferência é necessária para o funcionamento do serviço e ocorre com base no seu consentimento explícito.

8. Alterações nesta política

Caso esta política seja alterada de forma significativa, você será notificado no app na próxima abertura.
